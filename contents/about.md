### Notable Side-Quests

When I am not fighting with GTK widgets or rewriting this blog in increasingly obscure tech stacks, I build things like:

- **Distilled Reasoning Models**: My ongoing undergraduate thesis. I engineered a Python training pipeline for massive datasets and managed remote A100/H100 compute clusters to distill Llama-3.2-3B into an artifact that can run complex logic chains on edge-class hardware.
- **KMP S3 Backup Browser**: An internal tool built to stream and diff large `.dmp` files directly from DigitalOcean Spaces into a browser-based Monaco Editor. I locked it behind a Cloudflare Access Tunnel to achieve Zero-Trust security and keep everyone away from core server SSH access. 
- **Kelurahan Kemayoran Digital Portal**: A highly available web portal built under a strict Rp 0/month budget constraint. I paired Next.js (SSG) with Firebase and engineered a custom microservice using Cloudflare R2 to achieve zero-egress-fee image hosting.
- **Bin Detective**: A robust Express.js backend API deployed on Google Cloud Run using Docker, complete with GitHub Actions CI/CD to ensure auto-scaling reliability.
